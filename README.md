# Nuxt Template

This is a template for nuxt projects

## Setup

Make sure to install dependencies:

```bash
pnpm install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
pnpm dev
```

## Production

Build the application for production:

```bash
pnpm build
```

Locally preview production build:

```bash
pnpm preview
```

## Stack

- Nuxt 4
- Pinia
- Unocss
- @nuxtjs/i18n
